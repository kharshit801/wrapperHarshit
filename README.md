# Introduction


**FarmAI** is a cross-platform mobile application built with **React Native** and **Expo**, aimed at empowering farmers with AI and API-powered agricultural tools. The app offers real-time support for detecting diseases in crops and cattle, provides smart crop recommendations, weather insights, and access to market and soil health information â€” all in one unified platform.
